# stats-fetch

自动抓取 [国家统计局 data.stats.gov.cn](https://data.stats.gov.cn) 分省统计数据，支持多时间、多指标、多省份批量下载，结果导出为 Excel（明细表 + 透视表）。

---

## 环境要求

```bash
pip install requests pandas openpyxl
```

---

## 快速开始

**只需一行命令，全自动完成配置和抓取：**

```bash
python main.py
```

程序会自动检测配置文件是否存在：
- **不存在**：自动启动交互式配置生成器，引导你完成所有设置
- **存在**：直接使用现有配置开始抓取数据

---

## 详细使用说明

### 第一步：启动程序

```bash
python main.py
```

### 第二步：交互式配置（首次运行时）

如果是首次运行，程序会自动启动交互式配置生成器，按以下步骤操作：

#### 步骤 1/7：输入年份范围

```
============================================================
  步骤 1/7: 输入年份范围
============================================================
格式示例: 2015-2025
直接回车使用默认值: 2015-2025

请输入年份范围: 
```

- 输入你想要抓取的年份范围，格式为 `开始年份-结束年份`
- 例如：`2010-2025` 表示抓取2010年到2025年的数据
- 直接回车使用默认值 `2015-2025`

#### 步骤 2/7：选择一级目录

程序会自动从统计局网站获取所有一级目录列表：

```
============================================================
  请选择一级目录
============================================================
  [1] 国民经济核算
  [2] 人口
  [3] 就业人员和工资
  [4] 固定资产投资
  [5] 对外经济贸易
  ...
  [0] 退出/返回上一级
============================================================

请输入数字选择: 
```

- 输入对应的数字选择你想要抓取的指标大类
- 例如：输入 `1` 选择「国民经济核算」

#### 步骤 3/7：筛选有指标的二级目录

程序会自动检查你选择的一级目录下有哪些二级目录包含可用指标：

```
============================================================
  步骤 3/7: 筛选有指标的二级目录
============================================================
  提示: 正在检查该目录下哪些二级目录有可用指标...
  发现 X 个二级目录，开始筛选...
  
  [国民经济核算] 进度: 10% | 检查 [增加值] ... ✓ (3 个有效子目录)
  [国民经济核算] 进度: 20% | 检查 [指数] ... ✓ (5 个指标)
  ...
```

- 这一步会自动筛选掉没有可用指标的目录
- 等待筛选完成

#### 步骤 4/7：选择二级目录（可多选）

从筛选出的有效二级目录中选择：

```
============================================================
  请选择二级目录（可多选，如 1,3,5）
============================================================
  [1] 地区生产总值
  [2] 三次产业增加值
  [3] 人均地区生产总值
  ...
  [0] 全选
  [输入 q] 退出/返回上一级
============================================================

请输入数字选择: 
```

- **多选**：输入多个数字，用逗号分隔，例如 `1,3,5`
- **全选**：输入 `0`
- **退出**：输入 `q`

#### 步骤 5/7：为每个二级目录选择指标

为每个选择的二级目录选择具体指标：

```
------------------------------------------------------------
  处理二级目录: 地区生产总值
------------------------------------------------------------

  该目录有 5 个指标

============================================================
  请为 [地区生产总值] 选择指标（可多选）
============================================================
  [1] 地区生产总值
  [2] 第一产业增加值
  [3] 第二产业增加值
  [4] 第三产业增加值
  [5] 人均地区生产总值
  [0] 全选
  [输入 q] 退出/返回上一级
============================================================

请输入数字选择: 
```

- 同样支持多选、全选和退出

#### 步骤 6/7：选择省份

选择你想要抓取数据的省份：

```
============================================================
  步骤 6/7: 选择省份
============================================================
  [1] 使用预设省份组
  [2] 自定义选择省份
  [0] 退出

请选择: 
```

**选项 1：使用预设省份组**

```
============================================================
  请选择省份预设
============================================================
  [1] 长江经济带 (11个省份)
  [2] 沿海经济带 (11个省份)
  [3] 京津冀 (3个省份)
  [4] 长三角 (4个省份)
  [5] 全部省份 (31个省份)
  [0] 退出
============================================================

请输入数字选择: 
```

**选项 2：自定义选择省份**

可以从31个省份中自由选择多个省份。

#### 步骤 7/7：配置其他选项

最后配置抓取参数：

```
============================================================
  步骤 7/7: 配置其他选项
============================================================

请求间隔（秒，默认2，用于正式数据抓取）: 
输出文件名（默认: 统计局数据.xlsx）: 
```

- **请求间隔**：两次请求之间的等待时间，建议2-3秒，避免被封
- **输出文件名**：保存数据的Excel文件名

## 输出文件说明

程序会生成一个Excel文件，每个指标对应两个Sheet：

### 1. `指标名_明细` Sheet

原始记录，每行一条数据，包含以下列：
- **省份**：数据所属省份
- **年份**：数据年份
- **年份代码**：统计局内部年份代码
- **指标**：具体指标名称
- **数值**：指标数值
- **单位**：数据单位

### 2. `指标名_透视` Sheet

以年份为列的透视表，便于直接分析：
- 行：省份、指标、单位
- 列：各年份
- 值：对应的数据

## 命令行参数

| 参数 | 说明 |
|------|------|
| `python main.py` | 自动检测配置文件，不存在则启动交互式配置 |
| `python main.py --config my.json` | 使用指定的配置文件 |
| `python main.py --init` | 生成默认配置文件模板（不推荐，建议使用交互式配置） |
| `python main.py --list-provinces` | 列出所有预设省份组及代码 |
| `python main.py --no-human-like` | 禁用人类行为模拟（不推荐，可能被封） |
| `python main.py --no-interactive` | 禁用交互式配置（配置文件不存在时直接退出） |

## 配置文件说明

配置文件为JSON格式，示例：

```json
{
  "province_preset": "长江经济带",
  "custom_provinces": {},
  "time_range": "2015YY-2025YY",
  "delay": 2,
  "output": "统计局数据.xlsx",
  "indicators": {
    "地区生产总值": {
      "cid": "f401b4ba4e494f1e9e00629720f4408f",
      "indicatorIds": [
        "bd561031e16b47ad854697ad9e8d96ee",
        "3bb085e03ec04bbca3bb3aa807782d11"
      ],
      "rootId": "c4d82af16c3d4f0cb4f09d4af7d5888e"
    }
  }
}
```

| 字段 | 说明 |
|------|------|
| `province_preset` | 省份预设名称，如"长江经济带"、"全部省份"等 |
| `custom_provinces` | 自定义省份字典，格式为 `{"代码": "名称"}` |
| `time_range` | 时间范围，格式为 `开始YY-结束YY` |
| `delay` | 请求间隔（秒） |
| `output` | 输出文件名 |
| `indicators` | 指标配置，由交互式配置自动生成 |

## 常见问题

### 请求超时 / 连接失败

- 适当增大 `delay` 值（建议不低于2秒）
- 检查网络是否正常
- 统计局网站可能暂时不稳定，稍后重试

### 返回「无数据」

- 确认 `time_range` 中的年份范围该省份确实有数据
- 部分指标在早年份可能缺失属正常现象

### Excel 中没有透视表 Sheet

- 当同一省份 + 指标组合存在重复行时透视会失败
- 明细 Sheet 仍会正常保存

### 被网站封禁

- 确保启用了人类行为模拟（默认已启用）
- 增大 `delay` 值到3秒或更多
- 避免短时间内大量请求
